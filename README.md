## About

This repository showcases my PCB design experience in embedded systems and electronics.

Each project is designed in both **Altium Designer** and **KiCad**, allowing comparison between the two EDA tools while demonstrating transferable hardware design skills.

The goal of this repository is to document my PCB engineering journey and share reusable reference designs with the embedded systems community.
